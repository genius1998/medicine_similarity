2 OCR 결과 확인

추출된 제품명, 카테고리, 원료와 검토 필요 안내를 확인합니다.

3 원료 수정

잘못 추출된 원료를 삭제하거나 필요한 원료를 직접 추가합니다.

4 재추천 확인

수정된 원료 기준으로 유사 제품 TOP10을 다시 받아 비교합니다.
